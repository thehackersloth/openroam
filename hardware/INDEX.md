## Recommended Build Paths

### Minimal System (~$350)
Essential monitoring only:
- Power HAT
- Tank HAT
- Nav HAT (GPS only)
- Basic enclosure

### Standard System (~$650)
Full monitoring + basic control:
- All Core HATs
- Tank HAT
- Climate HAT
- Router HAT
- Camera Interface (rear only)

### Full System (~$1100)
Complete feature set:
- All modules listed
- Full camera system
- Radio interfaces
- AI processing

### DIY Budget Build (~$400)
Build from scratch using discrete components:
- All DIY BOMs
- Fabricate own PCBs
- 3D print enclosures
- Maximum learning, minimum cost

## Component Sourcing

### Primary Suppliers (US)
- [DigiKey](https://www.digikey.com) - Primary parts
- [Mouser](https://www.mouser.com) - Alternative source
- [Adafruit](https://www.adafruit.com) - Breakout boards
- [SparkFun](https://www.sparkfun.com) - Breakout boards

### Budget Sources
- [AliExpress](https://www.aliexpress.com) - Modules, connectors
- [LCSC](https://www.lcsc.com) - SMD components
- [eBay](https://www.ebay.com) - Surplus, modules

### PCB Fabrication
- [JLCPCB](https://jlcpcb.com) - Budget PCBs, assembly
- [PCBWay](https://www.pcbway.com) - Quality PCBs
- [OSH Park](https://oshpark.com) - US-made, excellent quality

### 3D Printing
- Own printer (FDM recommended)
- [Craftcloud](https://craftcloud3d.com) - Print service
- Local makerspaces
