# Dash Display Mount

Adjustable mount for 7" touchscreen displays with pod or double-DIN installation options.

## Features

- Fits standard 7" displays (800x480 or 1024x600)
- Tilt and swivel adjustment
- Pod mount or DIN cage option
- Integrated cable routing
- VESA 75 compatible back
- Quick-release mechanism

## Specifications

| Parameter | Value |
|-----------|-------|
| Display Size | 7" (165 x 100mm typical) |
| Mount Types | Pod, double-DIN, VESA 75 |
| External Dimensions | ~185 x 120 x 45 mm |
| Tilt Range | -15 to +30 degrees |
| Swivel Range | +/- 20 degrees |
| Material | PETG or ABS |

## Print Settings

| Setting | Value |
|---------|-------|
| Layer Height | 0.2mm |
| Infill | 30% |
| Perimeters | 4 |
| Material | PETG (black) |
| Supports | Yes (for ball joint socket) |
| Orientation | Back plate down |

## Hardware Required

| Item | Quantity | Notes |
|------|----------|-------|
| M4 x 8mm screws | 4 | Display to bezel |
| M4 x 12mm screws | 4 | VESA mount holes |
| M3 x 6mm screws | 4 | Back plate |
| M3 brass inserts | 4 | Heat-set |
| Ball joint 20mm | 1 | Tilt/swivel mechanism |
| Double-DIN cage | 1 | Optional DIN install |
| Adhesive cable clips | 4 | Cable management |

## Assembly Notes

1. Insert M3 brass inserts into bezel frame
2. Mount display in bezel using M4 screws
3. Attach back plate with cable routing
4. Install ball joint in socket (friction fit)
5. For pod mount: attach base to dash with adhesive/screws
6. For DIN mount: slide into standard double-DIN cage
7. Route HDMI and USB cables through back channel
8. Adjust viewing angle and tighten ball joint
9. Secure excess cable behind dash
