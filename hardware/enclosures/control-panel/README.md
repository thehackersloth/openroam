# Control Panel

Flush-mount switch panel for vehicle interior lighting, accessories, and system controls.

## Features

- Flush mount design for clean installation
- 6x rocker switch positions
- Integrated LED indicators
- Label window strips
- Modular switch cutouts
- Panel or surface mount

## Specifications

| Parameter | Value |
|-----------|-------|
| Switch Positions | 6 (configurable) |
| Switch Type | Standard rocker (19 x 13mm cutout) |
| External Dimensions | ~130 x 60 x 25 mm |
| Panel Cutout | 120 x 50 mm |
| Mounting | Flush panel or surface |
| Material | PETG or ABS |

## Print Settings

| Setting | Value |
|---------|-------|
| Layer Height | 0.16mm |
| Infill | 30% |
| Perimeters | 4 |
| Material | PETG (black or gray) |
| Supports | No |
| Orientation | Face down |

## Hardware Required

| Item | Quantity | Notes |
|------|----------|-------|
| Rocker switches | 6 | 19 x 13mm, LED type |
| M3 x 8mm screws | 4 | Panel mount |
| Spade terminals | 24 | 6.3mm for switches |
| 18AWG wire | 3m | Power distribution |
| Fuse holder inline | 1 | 15A blade fuse |
| Label paper | 1 | For switch labels |

## Assembly Notes

1. Test fit switches in cutouts (may need light sanding)
2. Print label strips and insert in windows
3. Snap switches into panel from front
4. Wire common ground bus to all switch grounds
5. Wire positive to each switch input via fuse
6. Connect switch outputs to loads
7. For flush mount: cut panel opening to spec
8. For surface mount: use included standoffs
9. Secure with M3 screws from behind panel
