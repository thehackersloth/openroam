# Camera Pod - Side

Mirror or fender-mounted camera housing for blind spot monitoring and lane change assist.

## Features

- Low-profile design to minimize drag
- Mirror arm or fender mount options
- Heated lens option for rain/fog
- Wide-angle fisheye lens support
- IP65 weatherproof rating
- Vibration-dampened mount

## Specifications

| Parameter | Value |
|-----------|-------|
| Compatible Cameras | IMX219, IMX708 (wide angle) |
| External Dimensions | ~30 x 25 x 22 mm |
| Viewing Angle | 120-180 degrees |
| Mounting | M5 stud or adhesive |
| Cable Exit | Bottom (concealed) |
| Material | ASA recommended |

## Print Settings

| Setting | Value |
|---------|-------|
| Layer Height | 0.15mm |
| Infill | 30% |
| Perimeters | 4 |
| Material | ASA (color match vehicle) |
| Supports | Yes (for mount boss) |
| Orientation | Flat side down |

## Hardware Required

| Item | Quantity | Notes |
|------|----------|-------|
| M2 x 4mm screws | 4 | Camera mounting |
| M5 x 15mm bolt | 1 | Mirror arm mount |
| M3 x 8mm screws | 2 | Cover plate |
| M3 brass inserts | 2 | Heat-set |
| Rubber grommet | 1 | Cable pass-through |
| 3M VHB tape | 1 | Adhesive mount option |

## Assembly Notes

1. Insert M3 brass inserts into body
2. Mount camera module with M2 screws
3. Route FFC cable through bottom grommet
4. Attach cover plate with M3 screws
5. For mirror mount: use M5 bolt through arm
6. For fender mount: clean surface and apply VHB tape
7. Allow 24h for adhesive cure before driving
