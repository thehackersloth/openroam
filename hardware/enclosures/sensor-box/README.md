# Sensor Box

Exterior-mounted enclosure for temperature, humidity, and air quality sensors.

## Features

- Radiation shield design for accurate temp readings
- Louvered sides for airflow
- Protected sensor cavity
- Weatherproof (IP44)
- Quick-release for calibration
- Mounting arm included

## Specifications

| Parameter | Value |
|-----------|-------|
| Compatible Sensors | BME280, SHT31, AHT20, PMS5003 |
| External Dimensions | ~50 x 40 x 60 mm |
| Sensor Cavity | 30 x 25 x 20 mm |
| Cable Entry | Bottom M8 gland |
| Mounting | Arm bracket, magnetic |
| Material | ASA (white recommended) |

## Print Settings

| Setting | Value |
|---------|-------|
| Layer Height | 0.2mm |
| Infill | 20% |
| Perimeters | 3 |
| Material | ASA white (heat reflection) |
| Supports | Yes (for louvers) |
| Orientation | Arm mount down |

## Hardware Required

| Item | Quantity | Notes |
|------|----------|-------|
| M8 cable gland | 1 | Bottom entry |
| M3 x 6mm screws | 4 | Sensor board mount |
| M3 brass inserts | 4 | Heat-set |
| M4 x 12mm screws | 2 | Arm attachment |
| Silicone conformal coat | 1 | Protect sensor PCB |
| 22AWG wire | 1m | I2C extension |

## Assembly Notes

1. Print in white ASA for solar heat reflection
2. Insert M3 brass inserts for sensor mounting
3. Apply conformal coating to sensor PCB
4. Mount sensor board inside cavity
5. Route I2C wires through cable gland
6. Attach radiation shield (outer louvers)
7. Mount using arm bracket - position away from heat sources
8. Avoid mounting above exhaust or near engine
