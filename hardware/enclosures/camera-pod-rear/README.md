# Camera Pod - Rear

License plate area camera mount with integrated IR illumination for reverse assist and recording.

## Features

- Flush mount design for license plate area
- Integrated IR LED ring for night vision
- Wide-angle lens support (170+ degrees)
- Weatherproof design (IP67)
- Reverse light trigger input
- Guide line overlay compatible

## Specifications

| Parameter | Value |
|-----------|-------|
| Compatible Cameras | IMX219, OV5647, dedicated backup cam |
| External Dimensions | ~45 x 25 x 20 mm |
| IR LEDs | 8x 850nm (invisible) |
| Viewing Angle | Up to 170 degrees |
| Mounting | License plate screws or adhesive |
| Material | ASA recommended (UV resistant) |

## Print Settings

| Setting | Value |
|---------|-------|
| Layer Height | 0.2mm |

## Hardware Required

| Item | Quantity | Notes |
|------|----------|-------|
| M2 x 4mm screws | 4 | Camera mounting |
| IR LEDs 850nm | 8 | 5mm through-hole |
| Resistors 47 ohm | 4 | For LED pairs (12V) |
| Cable gland M10 | 1 | Combined power/video |
| Silicone gasket | 1 | Lens seal |
| License plate screws | 2 | Standard 1/4-14 |

## Assembly Notes

1. Install IR LEDs in ring pockets, wiring in parallel pairs
2. Mount camera module with M2 screws
3. Apply silicone sealant around lens perimeter
4. Connect power (12V trigger from reverse light) and video cables
5. Thread cables through gland before final tightening
6. Mount using existing license plate screw holes
