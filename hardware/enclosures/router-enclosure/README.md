# Router Enclosure

Weatherproof enclosure for mobile router with integrated antenna mounts and ventilation.

## Features

- IP65 weatherproof rating
- 4x N-type antenna bulkhead mounts
- Passive ventilation with rain baffles
- DIN rail or panel mount options
- Power and ethernet cable glands
- Internal cable management

## Specifications

| Parameter | Value |
|-----------|-------|
| Compatible Routers | GL.iNet, Teltonika, Peplink |
| External Dimensions | ~180 x 120 x 60 mm |
| Antenna Ports | 4x N-type female bulkhead |
| Cable Entry | 2x M16 + 2x M12 glands |
| Mounting | DIN rail, panel, or strap |
| Material | ASA recommended |

## Print Settings

| Setting | Value |
|---------|-------|
| Layer Height | 0.2mm |
| Infill | 25% |
| Perimeters | 4 |

## Hardware Required

| Item | Quantity | Notes |
|------|----------|-------|
| N-type bulkhead | 4 | Female panel mount |
| M16 cable gland | 2 | Power and ethernet |
| M12 cable gland | 2 | Auxiliary cables |
| M4 x 8mm screws | 8 | Lid closure |
| M4 brass inserts | 8 | Heat-set in base |
| Silicone gasket cord | 1m | 3mm diameter |
| DIN rail clip | 2 | Optional mount |

## Assembly Notes

1. Insert M4 brass heat-set inserts into base
2. Install N-type bulkheads in antenna ports (use thread sealant)
3. Install cable glands (do not fully tighten yet)
4. Press gasket cord into lid groove
5. Mount router inside using adhesive standoffs
6. Route cables and attach connectors
7. Tighten cable glands around cables
8. Secure lid with M4 screws (do not overtighten)
9. Mount externally or in protected location
