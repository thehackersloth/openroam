# Antenna Mount

Roof-mounted antenna base for cellular, WiFi, GPS, and other communication antennas.

## Features

- 4-in-1 antenna base design
- Low-profile aerodynamic shape
- Weatherproof cable routing
- NMO mount compatible
- Magnetic or bolt-through mounting
- Integrated cable strain relief

## Specifications

| Parameter | Value |
|-----------|-------|
| Antenna Positions | 4 (configurable) |
| Mount Types | NMO 3/4", SMA bulkhead |
| External Dimensions | ~150 x 100 x 35 mm |
| Cable Exit | Rear or bottom |
| Mounting | Magnetic, adhesive, or bolt |
| Material | ASA (black) |

## Print Settings

| Setting | Value |
|---------|-------|
| Layer Height | 0.2mm |
| Infill | 35% |
| Perimeters | 5 |

| Supports | Yes (for internal cavity) |

## Hardware Required

| Item | Quantity | Notes |
|------|----------|-------|
| NMO mount 3/4" | 2 | Cellular/LTE antennas |
| SMA bulkhead female | 2 | WiFi/GPS antennas |
| Rubber roof gasket | 1 | For bolt-through mount |
| M6 x 20mm bolts | 4 | Bolt-through option |
| 25mm x 5mm magnets | 4 | Magnetic mount option |
| Coax cable RG316 | 2m | Low-loss runs |
| Butyl tape | 1 | Weather sealing |

## Assembly Notes

1. Install NMO mounts (hand tight, then 1/4 turn)
2. Install SMA bulkheads with lock washers
3. For magnetic mount: press-fit magnets into base cavities
4. For bolt-through: drill roof, install gasket, bolt down
5. Route coax cables through internal channel
6. Apply butyl tape around base perimeter
7. Connect cables to router antenna ports
8. Keep GPS antenna with clear sky view
9. Separate cellular and WiFi antennas by minimum 6 inches
