# Camera Pod - Mini

Compact 20mm cube camera housing for discreet interior or exterior mounting.

## Features

- Ultra-compact 20mm cube form factor
- Discreet appearance
- Ball joint mount for easy aiming
- Multiple mounting options
- Quick-release base
- Magnetic mount compatible

## Specifications

| Parameter | Value |
|-----------|-------|
| Compatible Cameras | IMX219 (standard/wide) |
| External Dimensions | 20 x 20 x 20 mm |
| Weight | ~8g (without camera) |
| Mounting | Ball joint, magnet, or adhesive |
| Cable Exit | Rear pigtail |
| Material | PETG or ASA |

## Print Settings

| Setting | Value |
|---------|-------|
| Layer Height | 0.12mm |
| Infill | 40% |
| Perimeters | 3 |
| Material | PETG (interior) / ASA (exterior) |
| Supports | No |
| Orientation | Lens facing up |

## Hardware Required

| Item | Quantity | Notes |
|------|----------|-------|
| M2 x 3mm screws | 2 | Camera board mount |
| Ball joint mount | 1 | 10mm ball |
| 6mm x 3mm magnets | 4 | Optional magnetic mount |
| FFC cable 50mm | 1 | Short run to adapter |
| Heat shrink | 1 | Cable strain relief |

## Assembly Notes

1. Mount camera board with M2 screws (countersunk)
2. Route FFC cable through rear slot
3. Snap cube onto ball joint base
4. For magnetic mount: press-fit magnets into base corners
5. Aim camera using ball joint, tighten set screw
6. Interior use: mount on dash, A-pillar, or headliner
7. Exterior use: seal FFC exit with silicone
